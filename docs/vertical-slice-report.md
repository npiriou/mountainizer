# Milestone report

## Successfully parsed

The supplied NTSC-U disc is indexed as ISO9660 volume `SSX3`; `SLUS_207.72` and `DATA/WORLDS/BAM.BIG` are detected and the complete ISO SHA-256 is recorded. The BIG archive's mixed-endian table is indexed and SDB/SSB/PHM/PSM members are extracted to a reusable read-only project cache.

SDB exposes 49 streaming areas, 183 chunk descriptors, and 159 subchunk descriptors matching the 159 SSB groups. Mountainizer maps those areas into all 17 playable courses and loads each course's event, shared-mountain, connector, and sky groups in one bounded SSB pass. Terrain, materials, MDR models, static instances, textures, splines, visibility curtains, particle/effect records, reference tables, and the spatial portion of camera-trigger tables are parsed into source-aware Core records. The generic fallback path retains type, track/RID, physical compressed group range, logical decompressed offset, byte preview, and diagnostics for unrecognized variants.

The local regression audit opens all 17 course assemblies. Every course contains terrain, props, models, and splines, every audited prop resolves to decoded MDR geometry, and no structured parser errors or generic fallback resources occur. Audit 72 uniquely validates all 41,113 Type-3 fixed headers, including exact self/track identities, exact Type-9 texture-bank subchunks with zero mismatches across 53,766 model-material references, zero reserved/runtime words, an unused invariant signed `-1` retail sentinel at `+0x90`, a bounded `0/1` wind-deformation halfword at `+0x92`, finite bounding spheres whose centers lie inside their world AABBs, and the single immutable collision-profile default/range (`-1` to resolved index 0, maximum known index 18), plus 61,282 PS2 DMA/VIF subprograms with 118,820 model references, 115,249 instance references, both SPR rewrite forms, and exact coverage of all 1,090,176 extension quadwords. All 115,249 VIF source blocks frame into 977,017 known commands and 4,112,542 per-instance RGBA5 vertex colors with no unknown commands or sequence violations; every color array exactly matches the count and contiguous VU destination of its model packet's terminal vertex array. Executable `0x0037e4d0` proves that the wind halfword gates an in-place render-matrix deformation callback for 1,203 simple visual instances across 90 models; 48 models occur in both states. The audit retains the Type-12 checks for 4,519 version-1 collision assets and 97 version-3 assets / 300 records, records that the v3 retained vector/matrix range is copied but retail-unused, and adds the complete Type-7 static-halo octree/queue/batch/occlusion/final-submission/state-compiler path for 1,679 records, including exact GS `ALPHA=0x48` source-alpha additive blending. All observed records now have typed containers, although many internal field and runtime semantics remain open research.

Audit 96 closes the optional Type-1 secondary terrain pass. The retail helper submits a separate GS-context-2 draw when render mask `0x0060` is present; bit `0x0040` selects alpha-table entry 17 and `ALPHA_2=0x58`, exactly `(Cs - 0) * Ad + Cd`. All 3,895 unique patches with a secondary RID use this destination-alpha additive path, and all 26,749 without one omit both enable bits. The always-present primary/lightmap combine remains the terrain-rendering semantic boundary.

Audit 97 closes that final combine boundary. Terrain setup `0x0038BCB4..0x0038BD78` installs primary selector 1 / `ALPHA_1=0x2A`, reducing to source replacement, and lightmap selector 8 / `ALPHA_2=0x81`, exactly `(Cd - Cs) * As`. All 623 unique Type-10 atlases are direct RGBA32 and total 6,984,704 pixels; raw PS2 alpha spans 0..255 with `0x80=1.0`. The older experimental multiplicative viewport blend was therefore the wrong operation: retail uses source-alpha-scaled subtraction/darkening.

Audit 98 closes Type-0's primary blend and optional texture-variant selection. Every shipped Type-9 state word has zero bits under renderer dispatch mask `0x00660000`; material bit `0x0008` either leaves selector 1 / `ALPHA_1=0x2A` (`Cs`) or injects `0x00040000` and selects selector 5 / `ALPHA_1=0x44` (`(Cs - Cd) * As + Cd`). The 2,575 unique materials split into 2,475 alpha-blended and 100 opaque records. Model-use registration initializes table selection as unsigned RNG output modulo the shared 2- or 5-entry frame count, proving the table supplies randomized per-use variants rather than timed animation.
